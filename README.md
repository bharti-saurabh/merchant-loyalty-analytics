# Merchant Loyalty Analytics

**Segment:** Merchant | **Category:** Customer Profiling | **Owner:** Straive Strategic Analytics | **Year:** 2024

## Objective
RFM-based customer segmentation to identify high-value customers, at-risk loyalists, and lapsed customers for targeted retention and reactivation campaigns.

## Methodology
1. RFM scoring (Recency, Frequency, Monetary) — quintile-based 1-5 scores
2. Segment assignment: Champions, Loyal, At Risk, Lost, New
3. Customer Lifetime Value (CLV) modelling
4. Cohort retention curve analysis
5. Share of Wallet estimation
